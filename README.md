# Vector Graphics Editor

Профессиональный графический редактор для создания и редактирования векторных фигур. Проект реализован с акцентом на объектно-ориентированное проектирование (ООП), расширяемость и чистую архитектуру.

## 🌟 Основные возможности

### Инструменты рисования
*   **Набор фигур:** Отрезок, ломаная линия, эллипс, прямоугольник, многоугольник.
*   **Интерактивный UI:** Отрисовка фигур в реальном времени с предпросмотром при движении мыши.
*   **Динамические плагины:** Поддержка добавления новых фигур (например, "Трапеция") без изменения основного кода через загрузку внешних модулей ([.dll / .jar]).

### Редактирование и трансформация
*   **Манипуляции:** Перемещение фигур (мышью или попиксельно клавишами), изменение размеров и вращение.
*   **Стилизация:** Настройка толщины линий, цвета контура и цвета заливки.
*   **Multi-select:** Групповое выделение объектов для массового редактирования или перемещения.
*   **Система слоев:** Создание, удаление, изменение порядка отображения слоев и перенос фигур между ними.

### Состояние и история
*   **Undo/Redo:** Полная история действий (Ctrl+Z / Ctrl+Y) для всех операций на холсте.
*   **Буфер обмена:** Возможность копирования и дублирования существующих фигур.

### Работа с данными
*   **Сериализация:** Сохранение и загрузка проектов в формате [JSON / XML / Binary].
*   **Масштабируемость:** Архитектура позволяет легко добавлять новые форматы сохранения.

## 🏗 Архитектура и паттерны

Приложение спроектировано с соблюдением принципов **SOLID**:
*   **Command:** Используется для реализации системы отмены и повтора действий.
*   **Factory / Prototype:** Для создания новых экземпляров фигур и их копирования.
*   **Plugin System:** Использование рефлексии (Reflection) для динамической подгрузки плагинов во время работы приложения.
*   **Dependency Inversion:** Основной функционал зависит от абстракций, а не от конкретных реализаций фигур.
